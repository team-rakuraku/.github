## Team Rakuraku - 라꾸채팅 SDK

<br/>

### 라꾸채팅 SDK란?
**라꾸채팅 SDK**는 샌드버드(Sendbird)와 같은 강력한 기능을 제공하면서도, 익명 채팅을 중심으로 사용자가 **라꾸라꾸(편안하게, 쉽게) 즐길 수 있는 채팅 경험**을 목표로 하는 채팅 SDK입니다.
