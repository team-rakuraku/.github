## Team Rakuraku - 라꾸채팅 SDK

### 라꾸채팅 SDK란?
**라꾸채팅 SDK**는 익명 채팅을 중심으로 사용자가 **라꾸라꾸(편안하게, 쉽게) 즐길 수 있는 채팅 경험** 전달을 목표로 하고 있습니다.
